1. Langkah pertama: Pembuatan Api Tungku: Nyalakan tungku tempa hingga bara api menyala secara merata dan bersuhu sangat tinggi.
  2. Langkah kedua : Pemanasan Logam: Masukkan besi ke dalam bara api. Pastikan logam terpapar api secara langsung dan merata. Tunggu hingga warnanya berubah menjadi merah menyala (oranye terang).
  3. Langkah ketiga : Penempaan (Forging):Gunakan penjepit untuk mengeluarkan logam yang sudah berpijar.Letakkan di atas landasan (paron) lalu pukul menggunakan palu secara berirama untuk membentuk besi sesuai rancangan (pipihkan, panjangkan, atau lengkungkan).
  4. Langkah keempat : Pemanasan Ulang: Logam akan mengeras dan sulit dibentuk saat mendingin. Jika logam mulai menghitam, masukkan kembali ke dalam tungku hingga memerah, lalu lanjutkan proses penempaan.
  5. Langkah kelima : Pembentukan Akhir: Lanjutkan proses penempaan detail dan penghalusan permukaan saat besi berwarna merah agak pudar (suhu turun) agar bentuknya presisi.
